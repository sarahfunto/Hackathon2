# Models

The trained model used in this project is **not stored in this GitHub repository**
because it exceeds GitHub file size limits.

## Hugging Face Model

The fine-tuned DistilBERT model is hosted on Hugging Face:

https://huggingface.co/sarahfunto/genetic-lora-merged

This model is automatically downloaded when running `main.py` using
Hugging Face Transformers.

## Why not stored here?

- Model file size > 100 MB
- GitHub file size limitation
- Best practice in ML projects is to host models separately
